Installation de Panda

Panda est écrit en Java.  Pour l'exécuter, il que Java soit installé sur votre ordinateur.

Visitez le site https://www.java.com pour installer Java

Ensuite, téléchargez le fichier panda.zip, dézippez-le, et doublie-cliquez sur le fichier panda.jar pour exécuter Panda.

Le logiciel Panda est disponible ici
