Théorème ∀x·x ∈ N => ∑_i=0;^x 2^i = 2^(x+1) − 1

Preuve par induction

Cas de base x=0

Montrons A[x:=0] = ∑_i=0;^0 2^i = 2^(0+1) − 1

  ∑_i=0;^0 2^i
=
  2^0
=
  1
=
  2^(0+1) − 1

Cas d'induction x > 0

HI A[x:=x-1] = ∑_i=0;^(x-1) 2^i = 2^((x-1)+1) − 1 = 2^x -1

Montrons A

  ∑_i=0;^x 2^i
=   { def. de ∑}
  (∑_i=0;^(x-1) 2^i) + 2^x
=   { HI }
  2^x -1 + 2^x
=   { arithmétique }
  2(2^x) -1
=   { def. de ^ }
  2^(x+1) -1
